# React Hooks Showcase

A minimal Vite + React + TypeScript project to demonstrate the usage of different types of React Hooks.

### Purpose

Showcase practical examples of React Hooks in a clean, lightweight setup.

### Getting started
1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Open the URL printed in your terminal.

### Available scripts
- `npm run dev` — Start development server with HMR
- `npm run build` — Type-check and build for production
- `npm run preview` — Preview the production build
- `npm run lint` — Run ESLint
